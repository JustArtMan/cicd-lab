# CI/CD Лабораторна робота

## Опис проекту
Демонстраційний проект для лабораторної роботи з CI/CD використовуючи GitHub Actions.

## Функціональність
- Простий калькулятор з основними операціями
- Допоміжні математичні функції
- Автоматичне тестування з покриттям коду
- CI/CD pipeline з GitHub Actions
